Html e Java Script

Html

HTML (HyperText Markup Language) é a linguagem de marcação padrão para a criação e estruturação de páginas da web. É usado para descrever a estrutura e o conteúdo de uma página, permitindo a formatação e organização de elementos como texto, imagens, links e outros elementos interativos. O HTML consiste em uma série de tags que são usadas para envolver e definir o significado dos elementos na página. Com o HTML, é possível criar páginas da web estáticas e dinâmicas, permitindo a criação de sites interativos e funcionais.

JavaScript

JavaScript é uma linguagem de programação amplamente usada no desenvolvimento web. Ela permite adicionar interatividade e dinamismo aos sites, possibilitando a criação de recursos como formulários interativos, animações, efeitos visuais, manipulação de elementos da página e muito mais. JavaScript é executado no lado do cliente, ou seja, no navegador do usuário, e é uma das principais tecnologias utilizadas para criar experiências interativas na web.
